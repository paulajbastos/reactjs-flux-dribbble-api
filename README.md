# reactjs-flux-dribbblr-api
This is a ReactJs APP made with Flux to connect Dribbble API.

Instructions:


:: Install NodeJs

:: Install Ruby (if windows)

:: Install Grunt Cli

:: run 

$ npm install 

:: to development APP run

$ grunt

:: to build APP run

$ grunt build


DEMO: http://dev.paulajbastos.com/reactjs-flux-dribbble-api/
